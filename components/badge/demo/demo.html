Badge 标记

出现在按钮、图标旁的数字或状态标记。

基本用法
文字

展示新消息数量。

代码演示
                                
<div>
    <span class="badge-bg">
        <em class="badge-text"></em>
        <i class="icon-badge" count="1"></i>
    </span>
    <span class="badge-clear">
        <em class="badge-text">文字</em>
        <i class="icon-badge" count="1"></i>
    </span>
</div>
封顶数字

可以自定义最大值。

代码演示
                                
<div class="badge-dynamic">
  <span class="badge-bg">
      <em class="badge-text"></em>
      <i overflow="9" count="10" class="icon-badge"></i>
  </span>
  <span class="badge-bg">
      <em class="badge-text"></em>
      <i overflow="99" count="100" class="icon-badge"></i>
  </span>
  <span class="badge-bg">
      <em class="badge-text"></em>
      <i overflow="999" count="1000" class="icon-badge"></i>
  </span>
</div>
小红点
文字

以红点的形式标注需要关注的内容。

代码演示
                                
<div class="badge-point">
    <span class="badge-bg">
        <em class="badge-text"></em>
        <i class="icon-badge-point"></i>
    </span>
    <span class="badge-clear">
        <em class="badge-text">文字</em>
        <i class="icon-badge-point"></i>
    </span>
</div>
独立使用

不包裹任何元素即是独立使用，可自定样式展现。

代码演示
                                
<div class="badge-bubble">
    <i class="icon-badge" count="9" overflow="99"></i>
    <i class="icon-badge" count="100" overflow="99"></i>
    <i class="icon-badge-clear icon-badge" count="999" overflow="999"></i>
</div>

API

类别 属性名 类型 默认值
消息数量 count Number
封顶数量 overflow Number 99
