Breadcrumb 面包屑

显示当前页面在系统层级结构中的位置，并能向上返回。

基本用法
  • 首页/
  • 联系信息/
  • 通讯录

适用广泛的基础用法。

代码演示
                                
<ul class="jm-crumbs">
    <li><a href="">首页</a><i>/</i></li>
    <li><a href="">联系信息</a><i>/</i></li>
    <li class="active">通讯录</li>
</ul>
带图标的
  • 首页/
  • 联系信息/
  • 通讯录

图标放在文字前面。

代码演示
                                
<ul class="jm-crumbs">
    <li><a href=""><span class="jm-icon jm-icon-detail"></span><span>首页</span></a><i>/</i></li>
    <li><a href=""><span class="jm-icon jm-icon-detail"></span><span>联系信息</span></a><i>/</i></li>
    <li class="active"><span class="jm-icon jm-icon-detail"></span><span>通讯录</span></li>
</ul>
分隔符
  • 首页>
  • 联系信息>
  • 通讯录

可以自定义分隔符。

代码演示
                                
<ul class="jm-crumbs">
    <li><a href="">首页</a><i>></i></li>
    <li><a href="">联系信息</a><i>></i></li>
    <li class="active">通讯录</li>
</ul>
