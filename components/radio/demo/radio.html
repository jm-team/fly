Radio 单选框

在一组备选项中进行单选。

基础用法

由于选项默认可见，不宜过多，若选项过多，建议使用 Select 选择器。

代码演示
                                
<label class="jm-radio-box" for="">
    <span class="jm-radio-icon-box">
        <span class="jm-icon"></span>备选项
    </span>
    <input type="radio" class="jm-radio" checked name="options">
</label>
<label class="jm-radio-box" for="">
    <span class="jm-radio-icon-box">
        <span class="jm-icon"></span>备选项
    </span>
    <input type="radio" class="jm-radio" name="options">
</label>
