分页

基本用法

具体使用方法：

给select设置属性model绑定数据，需要将model设置.sync实现数据的双向绑定，否则在改变状态时，使用者的数据并没有变化。

单选时，model 只接受字符串和数字类型，多选时只接受数组类型，组件会自动根据option的value来返回选中的数据。

可以给select添加 style 样式，比如宽度。

在展开选择器后，可以使用键盘的up和down快速上下选择，按下Enter选择，按下Esc收起选择器。

代码演示
                                
laypage({
    cont:'page1',
    pages:100,
    first:1,
    last:100
});
页码显示的个数

展示的页码个数

代码演示
                                
laypage({
    cont:'page2',
    pages:100,
    first:1,
    last:100,
    groups:7
});
不显示首尾页

不显示首尾页

代码演示
                                
laypage({
    cont:'page3',
    pages:100,
    first:false,
    last:false
});
只显示上一页和下一页

只显示上一页和下一页，不显示页码

代码演示
                                
laypage({
    cont:'page4',
    pages:100,
    first:false,
    last:false,
    groups:0
});
跳转到具体页码

跳转到具体页码

代码演示
                                
laypage({
    cont:'page5',
    pages:100,
    first:1,
    last:100,
    skip:true
});
页码改变的事件回调

页码改变的事件回调

代码演示
                                
laypage({
    cont:'page6',
    pages:100,
    first:1,
    last:100,
    jump:function(obj){
        if(obj.curr != 1){
            alert(obj.curr)
        }
    }
});

    pages:100,
    first:1,
    last:100,
    prev:'',
    next:''
});
