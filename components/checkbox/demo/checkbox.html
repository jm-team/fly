Checkbox 多选框

在一组备选项中进行多选。

基础用法

单独使用可以表示两种状态之间的切换。

代码演示
                                
<label class="jm-checkbox-box" for="">
    <span class="jm-checkbox-icon-box">
        <span class="jm-icon"></span>备选项
    </span>
    <input type="checkbox" class="jm-checkbox" checked="checked" name="isMale1">
</label>
<label class="jm-checkbox-box" for="">
    <span class="jm-checkbox-icon-box">
        <span class="jm-icon"></span>备选项
    </span>
    <input type="checkbox" class="jm-checkbox" name="isMale2">
</label>
